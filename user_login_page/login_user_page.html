USER LOGIN PAGE

Lab Practical -3

1. Objective: To create a simple login page using html

2. Theory: The <form> element is used to create the form, and the action attribute which tells     the browser where to send the form data when the user submits it. The method attribute is     set to "post" which tells the browser where to send the data in the body of the HTTP request.

    The <label> element is used to create a test label for the input fields, and the for attribute     is used to associate the label with the corresponding input field. The <input> elements are     used to create the test fields for the username and password. The type attribute is set to     "test" for the username field and "password" for the password field, which tells the browser     to mark the characters entered in the password field.

    The required attribute is added to the input fields, which tells the browser that these fields     must be filled out before the form can be submitted.

    The <input type="submit"> element is used to create a submit button, and the value attribute     is set to "Login" which will be displayed on the button.

    There are several other inputs such as radio and checkboxes.
